# Next.js Rules

Use:

- App Router
- Server Components by default
- Server Actions when appropriate
- Metadata API
- Suspense
- Dynamic imports

Avoid:

- Unnecessary use client
- Deprecated APIs
- Client-heavy pages

Performance:

- Optimize images
- Lazy load heavy sections
- Minimize client-side JS

SEO:

- Every page must have metadata
- Use semantic HTML
- Optimize Core Web Vitals
